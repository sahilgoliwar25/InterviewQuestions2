Q1. Explain br tags.

Ans. br tag in HTML is a special tag which is used to create the break in line. It is an empty tag and doesn't need any closing tag. br tag is written as <br />.


Q2. Explain how many different types of lists we have in html.

Ans.There are three types of listing tags in HTML namly ol(Ordered List), ul(Unordered List) and dl(Description List). Listing tags helps to create different types of lists for various purpose. With ol and ul tags we use li tag for listing every element of the list.

ol(Ordered List)- This tag creates the sequence of lists using the type specified in the tag. There are 5 ol listing types: 1(numeric), a(small alphabet), A(large alphabet), i(small roman) & I(large roman). Deafult listing method of ol tag is 1(numberic).

ul(Unordered List)- This tag creates the non-sequence of lists using the type specified in the tag. There are 4 ul listing methods: none, disc, circle & square. Deafult listing method of ul tag is disc.

dl(Description List)- This tag is used to represent the description list. This tag is used with dt and dd tag where dt is used to specify the description list and dd stands for the definition description.


Q3. What are semantic tags.

Ans. Semantic tags in HTML are those tags which define the meaning of the content they contain. They are also called as naming tags. They helps us to make our code more readable. Some of Semantic tags are form, header, footer, table, etc.
